# CRUD Next to the platform

using next.js to create a CRUD application with forms only.
no state, can disable javascript high performance.

#Instalation

```shell
$ git clone && yarn
```

#Running Mock Server

```shell
$ yarn mock-server
```

#Running Next Dev

```shell
$ yarn dev
```

#Build and Run

```shell
$ yarn next build
$ yarn next start
```
